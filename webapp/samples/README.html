这里的例子演示了Getty向脚本中注入变量的用法，使用这些变量可以获取Web输入、向Web页输出文字，或者从Session、Cookie中获取你想要的信息。

运行param.groovy页面时请在URL后面加上一个name参数，例如param.groovy?name=jacky，这个页面向您演示name变量已经自动注入到了脚本中，这样的写法是不是很方便。

如果你对MVC感兴趣，mvc.groovy演示了最简单的MVC框架，在_model容器中添加一个对象，使用_view变量指向一个Java Server Page，就实现了一个最简单的MVC框架。

JSON输出也可以，info.groovy和stat.groovy中都演示了开发过程有多么简单。
