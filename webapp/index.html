Groovy on Jetty!

------------------------------------

This page is used to test the proper operation of the Getty server after it has been installed. If you can read this page it means that the Getty server installed at this site is working properly.

You may now add content to the directory webapp/. Note that until you do so, people visiting your website will see this page and not your content.

You can write your content in Groovy, see samples. Groovy helps you build your site quickly. Because it runs in JVM, you can use components writed in Java.

For information on Getty please visit the Getty website on Github.
